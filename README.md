# 🍰 Pastelería Un Poco Dulce - Web Project

¡Bienvenido al repositorio de Un Poco Dulce! Este es un proyecto de desarrollo web Front-End. El sitio está diseñado para ofrecer una experiencia visualmente rica, destacando productos mediante un diseño elegante y moderno.

## 🚀 Vista General

El proyecto consiste en un sitio web multipágina (MPA) que permite a los usuarios conocer la historia de la pastelería, explorar las modalidades de pedido, registrarse e iniciar sesión, y contactar directamente al negocio.

## 📸 Características Principales

· Diseño Responsivo: Estructura adaptativa utilizando Flexbox para garantizar que el contenido luzca bien en diferentes dispositivos.

· Carrusel 3D Interactivo: Galería de imágenes en la sección "Quiénes Somos" con efectos de transición y expansión (flex-grow) al pasar el cursor.

· Sistema de Formularios: Interfaces completas para Registro, Login y Contacto con validación nativa de HTML5.

· Estética Coherente: Paleta de colores cálidos (naranjas y tierras) combinada con transparencias oscuras para una lectura cómoda y un look profesional.

· Integración de Mapas: Uso de iframe para la ubicación física del local en la sección de Pedidos.

## 🛠️ Tecnologías Utilizadas

· HTML5: Marcado semántico para mejorar el SEO y la accesibilidad.

· CSS3: Estilos personalizados, animaciones de transición y manejo de layouts complejos con Flexbox.

· Google Fonts: Integración de tipografías premium como Mystery Quest para títulos y Elsie Swash Caps para cuerpos de texto especiales.
